## 프론트엔드 개발자 지원자용 실무 과제
### 기본요건
1. 데이터 저장을 위해서 Local/Session Storage등 자유로이 사용 가능
1. 프로젝트는 Next.js (14.x.x ~ 15.x.x)로 구현
2. 스타일링은 Tailwind CSS (3.x.x)를 사용
3. 프로젝트는 반드시 Typescript를 사용
4. 라이브러리는 자유롭게 사용
